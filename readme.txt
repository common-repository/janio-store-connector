== Description ==
This plugin will allow you to seamlessly ship orders received on your WooCommerce store with Janio without the need to manually import order details.

Before you start the integration process, please make sure that:

1. You have an active Janio merchant account.
2. You have admin rights access to your WooCommerce store.
